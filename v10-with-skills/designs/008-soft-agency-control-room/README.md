# Soft Agency Control Room

## Thesis

This prototype frames WordPress Playground like a quiet production control room. The live iframe sits in a double-bezel preview tray, while a floating island toolbar provides refresh, address, save, WP Admin, Homepage, and focus. One secondary control room panel owns launch routes, saved playgrounds, blueprint gallery, Site Manager tabs, export actions, and destructive reset.

## Skill material used

Used `taste-skill/SKILL.md` for anti-slop defaults: single accent, tactile controls, visible loading/error/empty states, and stronger layout judgment. Used `soft-skill/SKILL.md` for double-bezel surfaces, soft structuralism, fluid island controls, nested button behavior, premium spacing, and restrained motion. Used `redesign-skill/SKILL.md` to avoid duplicate feature entry points and generic card-heavy output.

## Flow model

The first viewport is the running session. The island toolbar is the only persistent command row, and the control room panel is the only secondary surface. Launcher, Manager, and State are mutually exclusive panel modes, preserving the v7 lessons around iframe primacy, one secondary surface, explicit focus mode, and no two-row top bar.

## Trade-offs

The soft style uses more spacing than a dense developer cockpit, so the blueprint gallery scrolls inside the secondary panel. In return, the destructive Apply+Reset model and save state are easier to notice. Mobile keeps the island controls touch-friendly and moves the control room below the iframe.
