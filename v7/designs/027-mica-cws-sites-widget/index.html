Primitive composition map

W11
Headline. Mica material applied to every chrome surface as CSS tokens (--mica-base, --mica-card, --mica-sidebar) — sidebar, top capsule, CWS, panel, library overlay, mobile sheet, mobile bar. Every surface uses backdrop-filter: blur(40px) saturate(180%). Five-row Settings.app sidebar is the IA frame.
CWS
Co-headline. A separate Mica capsule pinned to the top-left corner, persistent across every panel (A · D · E · F · G · H · M), showing live status (name + dot + version line). It's the sole door to Sites/saved/new/library. Expands DOWNWARD from its corner anchor with a Fluent corner pin.
VEN
Restraint. Sidebar has exactly 5 rows, Site Manager has 5 tabs, Sites widget has 3 segments. ≤5 rule respected.
DEV
Slim bar. Top capsule is a single 40px row holding refresh, address, save-pill, Fullscreen, More. No second row. Never doubles as nav.
MOB
Parity. Same IA on mobile: sidebar rotates 90° to a 5-cell bottom Mica nav, CWS becomes a top capsule still pinned to the top, secondary content opens as a bottom sheet.

Banned-pattern check

  • Two-row top bar? No — single 40px capsule.DROP avoided
  • Depth-as-interactive-control? No — Mica blur is uniform across surfaces, only used for hierarchy.DROP avoided
  • Dual-channel duplication (visual UI + terminal echo)? No — no terminal anywhere; Logs is a one-way reader.DROP avoided
