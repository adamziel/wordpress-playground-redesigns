∎ · LEGEND Modes & primitives at a glance Two modes, one shared door. The CWS is both a status display in atlas mode and the canonical door home in site mode.
Two modes
Atlas mode
Pan/zoom spatial canvas of saved playgrounds. Default at idle. The CWS shows the running unsaved playground (if any).
Site mode
Iframe fills the canvas. CWS becomes the door home + sub-doors. Press ⇧A or click CWS Atlas door to return.
CWS · two states · one widget
Status mode (atlas)
name + state + tiny minimap. Click → focus the running site.
Door mode (site)
name + state + minimap + 4 sub-doors (Manage / Library / Logs / Atlas).
Expanded panel
CWS blooms in place into 380×560 panel. Sub-tabs: Your sites · Start new · More.
3LD · visual only
z0 = iframe / atlas tile (sharp)
never blurred — content has primacy
z1 = atlas behind CWS panel (8px blur)
decorative depth only
z3 = library overlay scrim (36px)
never an interactive control surface
Affordance budget
Idle = 8 affordances
1) refresh · 2) URL · 3) status pill · 4) Library · 5) New playground · 6) Zoom cluster · 7) Fullscreen · 8) CWS widget. The 8 atlas tiles are content, not chrome.
Site = 6 affordances
topcap (refresh + URL + status) · fullscreen pill · CWS (one surface containing the 4 sub-doors).
