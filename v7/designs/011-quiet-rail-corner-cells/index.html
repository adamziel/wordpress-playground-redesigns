J How the composition is read QR provides the IA backbone (one rail, one pane). CWS provides the cell-as-widget: live status without spending an extra affordance.
Sites cell
Green pip = at least one saved site, never destructive. Stat: "3 saved." Click opens the Sites pane (Your / New / Library).
Manage cell
Blue pip when the active site is healthy. Stat: PHP version. Click opens Site Manager with the 5-way tabbed segmented control.
Library cell
Purple pip = library is fresh. Stat: count of blueprints. Click jumps the Sites pane directly to the Library sub-tab.
Logs cell
Amber pip when warnings; red on fatal. Stat: warning count. Click opens Manage → Logs and pre-filters by level.
