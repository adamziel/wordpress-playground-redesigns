The frame URL pill and Fullscreen button live inside the current pane, not in idle chrome — they belong to whichever category is active. The momentum gauge is passive (not interactive). The seam is a 1px visual hairline. Title-bar traffic lights are window-chrome, not Playground chrome.

Smart-resize detents (six snaps)

  • 56 / 1384 — iframe focus (sidebar collapsed to icon rail)
  • 280 / 1160 — rest (Ventura default sidebar)
  • 432 / 1008 — sidebar focus (30% — clicking any category)
  • 720 / 720 — equal weight (editor / log work)
  • 1008 / 432 — sidebar primacy (long-form authoring)
  • 1440 / 0 — fullscreen iframe (F or Esc to exit)

Primitives composed

VEN — sidebar IA + tinted glyph categories with distinct silhouettes (play / grid / sliders / file-code / terminal — not five tinted squares). SR — sidebar↔iframe seam is the smart-resize surface; whichever side you touch grows; 6 snap detents; momentum gauge etched at the seam. + honest scoping on 3LD (used only as decoration on the receding pane via `.dim` filter) and GLS (only on the Library overlay scrim).
