005 DevTools dock · smart resize SR + DEV + (CWS micro)

A slim DevTools top bar carries the URL, name + status-pill, save, fullscreen, and a kebab — nothing else. Below it, the screen is a single smart-resize between the iframe (left) and a DevTools-style dock (right) that swaps Sites, Settings, Files, Blueprint, Database, and Logs. Whichever pane you interact with grows; rest is 60/40, dock focus 35/65, iframe focus 90/10 (dock collapses to a vertical icon strip but stays a door). The seam shows a momentum gauge with the current ratio. One row of chrome, one secondary surface, one door per feature.
