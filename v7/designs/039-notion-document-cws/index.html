Notion Document · Corner Widget Sites

Each playground is a Notion-style document. The iframe is one block. The blueprint is one block. Files, database, logs, settings — each is a block in the same document. You scroll the playground; there is no Site Manager pane and no top toolbar. The only persistent chrome is a single frosted corner widget at the top-left that doubles as live status of the current document AND is the door to all other playgrounds, the library of 43 blueprints, and the six "Start new" sources. Iframe-block focus mode (F) reflows the document around the iframe; full-iframe promotion (Esc to exit) hides everything else.

Iframe block · focus mode (F)

desktop · 1440×900 · iframe block expands, surrounding blocks reflow narrower

Pressing F on the iframe block (or clicking the Focus button in its hover toolbar) is the intermediate state between embedded and fullscreen. The iframe widens beyond the document column (negative margins), the column re-flows, and the surrounding blocks remain reachable by scroll — single keystroke, soft animation, no chrome added.
