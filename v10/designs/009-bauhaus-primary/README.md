# 009 — Bauhaus Primary

## Thesis

WordPress Playground rendered as a strict Bauhaus composition. The chrome is a Dessau-school grid of primary red (#E74C3C), primary yellow (#F1C40F), and primary blue (#2C7BE5), laid out on a bright off-white field (#FAFAFA) and bound by pure-black 3px rules. Typography is Inter / Helvetica Neue geometric, no italics, no curves except where a circle is the icon. Every IA destination is given a geometric body: Sites is a red square, Settings is a yellow triangle, Files is a blue circle, Database is a black bar, Logs is a red dot. The five chips line up on the left rail like the colored sliders of an analog mixer, and each one swaps a single, dedicated pane that reads like a poster.

## What's locked vs. explored

The v7 #010 interaction model is locked: one quiet rail, one swap pane, one slim top capsule (here a hard-edged black slab, no radius), one corner site widget (here rendered as a primary-color flag with the site name stenciled across it), a z3 modal Library overlay (here a grid of primary-colored blueprint blocks), and a mobile mirror with the rail rotated to a bottom tab bar. What's explored is the visual idiom and the iconography: the rail icons are not glyphs but pure geometric primitives, the borders are heroically thick (3px black), nothing is rounded, and the iframe wallpaper layer is the only soft surface in the composition. The pane arrangement is a strict 5-dock — one rail icon, one pane mode — to honor the Bauhaus rule of one form per function.

## Trade-offs

The thick black borders trade subtlety for unmistakable structure: at a glance you can label every panel. The bright primaries are loud, but the iframe is allowed to be louder still because it occupies the largest unbroken white area in the composition. The Apply+Reset confirmation uses both a typed phrase ("RESET PLAYGROUND") and an auto-snapshot banner; the diff sheet is presented as a two-column block list rather than a fancy table. The visual idiom is unmistakable from a single screenshot — which was the brief.
