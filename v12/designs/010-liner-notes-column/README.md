# 010 — Liner Notes Column

## Thesis

A 1965 jazz LP gatefold liner-notes column. Single column, single ink, a typewriter-tight track-listing. The Playground adopts that listing instinct — every saved playground is a numbered track on a side, the title runs in Newsreader italic, leader dots fill the slack, and a tabular-figure runtime sits on the right. Runtime is creatively used for "minutes since opened." The page reads as a record sleeve, not a console.

## List pattern

**Track-listing column.** Each entry is a single row: a tabular track-number (`01`, `02`, `03`, `04`), an italic title, a leader-dot run that fills the available width, and a right-aligned runtime in `MM:SS` for saved sites (or a short keyword like `stable`, `trunk`, `editor` for start-routes). No icon. No metadata second row. One column, hairline-ruled between rows. The active track wears a steel-blue ▶ in the left margin and a steel-blue runtime; the unsaved playground is in italic amber. Start routes share the same dialect under a "Side B — start a Playground" section title. The Blueprint Library overlay uses the same pattern in two columns under four "Sides" (A, B, C, D), one per category cluster.

## Focus test

Look at the pane for three seconds. The track with a small ▶ in the margin is the cut you're playing. The amber-italic row is the unsaved playground. Track numbers and runtimes align by tabular figures down the column — Source-Serif-style tnum on the body font, not a monospace face — so the page reads as a single coherent listing.

## Trade-offs

The runtime column is a metaphor: "minutes since opened" is meaningful only if the user has been in the Playground for a while. For start routes, we substitute keyword runtimes (`stable`, `trunk`, `editor`) which keeps the column shape consistent but slightly bends the metaphor. The leader-dot pattern is generated by overflow-hidden CSS rather than a true OpenType leader, which means very long titles will collide with the runtime rather than truncate — production would use a `text-overflow: ellipsis` on the title. The five rail marks (▶ ▣ ▤ ▥ ▦) are Unicode block symbols; they were chosen so the silhouettes are unmistakably different, but accessibility production should swap them for SVG with descriptive labels.
