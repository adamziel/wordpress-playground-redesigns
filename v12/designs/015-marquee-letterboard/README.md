# 015 — Marquee Letterboard

## Thesis

An independent cinema at 10pm. The Playground browser is the lobby letterboard: a black grooved board of plastic Anton letters, each entry a single line that reads as one half-poetic film slot — `COFFEE SHOP — NOW SHOWING`. The state is not a row of meta; it is a marquee tagline. The whole interface speaks the same dialect: bulb yellow only on the saving pill and the active dock's glow; everything else is matte black plastic with white letters slotted in.

## List pattern

Marquee letterboard. Each saved site is a single horizontal slot — title in Anton uppercase, then an em-dash, then one slogan-style state word: `NOW SHOWING`, `IDLE`, `PREMIERE`, `MATINEE`, `LATE SHOW`. Start routes follow the same dialect: `VANILLA WORDPRESS — TONIGHT`, `IMPORT .ZIP — DOUBLE FEATURE`. The library overlay is the full marquee wall — 16+ blueprints listed as features of the week.

## Focus test

The iframe is the loudest object. The pane is a column of marquee lines; the rail is five Anton letters; the top capsule is the brass bar above the iframe doorway. The user reads the pane as a single typographic frieze. The bulb-yellow accent appears exactly on the saving pill and on the active letter's glow — no other yellow on the page.

## Trade-offs

A single state word can't carry rich meta — there is no version, no size, no last-opened time on the marquee face. We accept this as the cost of the metaphor. Hover surfaces the full meta in a small pop-card. Mobile collapses the marquee to a smaller plate but keeps the title-em-dash-tagline form so the reading rhythm survives.
