# 011 — Inscription Hall

## Thesis

A memorial hall, at midnight, lit by a single picture light. Each saved playground is a stone tablet on a low wall — chiseled Roman capitals, a Roman numeral date, a bronze hairline rule. Nothing else. The metaphor is gravitas: the act of saving a Playground is treated with the dignity of an inscription, not the chatter of a dashboard row. The room is quiet; the iframe inside it is what matters.

## List pattern

Inscription. Saved sites and start routes are rendered as single chiseled Cinzel lines centred in the pane — `COFFEE SHOP · MMXXVI`, with a 1px bronze hairline rule beneath. No icon. No second line. The Roman numeral date is the only datum the inscription carries; everything else (WP, PHP, size) is reserved for a wall-cartouche hover-state that never violates the surface. The library overlay continues the same dialect: 16+ blueprint titles inscribed vertically as a hall of names grouped by category.

## Focus test

The iframe occupies 65% of the desktop frame. The rail is 70px of cold stone. The pane is a hairline column of inscriptions on a slightly warmer stone. The marquee top capsule reads as the brass plaque above the iframe doorway. If you crop the screenshot to the iframe only and the inscription pane only, you see two surfaces of equal calm — the iframe loudest by content, the pane loudest by typography.

## Trade-offs

Inscriptions defer the meta data — a user wanting "how big is this site?" must hover a tablet to see the cartouche. This is the deliberate cost of the metaphor: legibility of language above legibility of metric. The Cinzel face is reserved for titles and the rail numerals; body chrome uses Atkinson Hyperlegible at small sizes so working text never collides with the inscriptional register.
