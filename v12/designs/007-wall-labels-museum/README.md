# 007 — Wall Labels (Museum)

## Thesis

A small natural-history gallery after closing. Dark walls. A single picture light on each object. The interface adopts the etiquette of museum signage: every saved playground is given a typeset wall-label with a title in Spectral italic and a single prose ascription sentence underneath. The label reads as language; it does not file the site as a data row.

## List pattern

**Wall-label cartouches.** Each saved playground entry has a Roman numeral in the left margin (i, ii, iii, iv), then a title in Spectral italic 18px, then one full sentence that names what the object actually is — for example: "A WooCommerce storefront with a Storefront-coffee theme, twelve sample products, twelve days in residence. WordPress 6.7 on PHP 8.2. Last edited Friday morning." There is no second row of values. There is no icon. The data is hidden inside the prose. The active label wears a soft bone-cream picture-light cone above it. The start-routes panel uses the same cartouche pattern (a — f) with a sentence that explains what the route does. The Blueprint Library uses the same pattern in a 2-column wall.

## Focus test

Look at the pane for three seconds. The only label wearing a picture-light cone is the active one. The italic-warm label is the unsaved playground. Every other entry is a paragraph, indistinguishable in its right to attention from its neighbours — which is the point. The room is dim; the picture light does the navigation work.

## Trade-offs

Prose ascription is slow to read. A user scanning for a particular saved site will spend longer than they would scanning a data row. We accept this in exchange for a list shape that nobody else in v12 will produce. Long sentences truncate gracefully because they are sentences — they fold without being maimed. The ascription sentence is hand-written per site (and per blueprint) rather than auto-generated from a meta object; production would template these. The metaphor breaks slightly on mobile, where the label paragraph wraps narrower; we keep the single picture-light cone but shrink the typography rather than collapse the prose.
