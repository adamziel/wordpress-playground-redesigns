v12 · twenty distinct dark pages

Twenty list-shapes, not twenty colors of the same row.

v11 was rejected for visual sameness — every design rendered saved sites and start routes as the same icon-left + two-rows-of-text card. v12 fixes the underlying failure: each of the twenty designs invents a distinct list-rendering ontology — ledgers, departure boards, postcards, stamps, book spines, wall labels, hand-lettered ledgers, lookbooks, liner notes, inscriptions, variable-axis specimens, stencil plates, size-as-data hierarchies, marquee letters, plan-view rectangles, jersey numerals, broadsheet mastheads, Bodoni hairlines, tabular figures. Same locked interaction model, same dark constraints. Twenty different things to look at.

020

Tabular Figures Column

Open Notes

A treasurer's annual report in dark mode. Saved sites become a single column — title left, right-aligned tabular figures right. Figures align by feature, not by glyph width.

List pattern
tabular figures column — each entry is a single grid row: title left in Source Serif 4 16px, then right-aligned columns of Source Sans 3 figures (`6.7 8.2 14.2 MB 22 May`) with `font-variant-numeric: tabular-nums lining-nums`. The figures align because they're tabular, not because the type is monospace. No header row repeated per item, no per-row icon, no meta-second-row.
Accent
drafting-blue #5C8AA6 — saving pill, active row underline, active dock tick, identity tint. Single accent.
Type
Source Serif 4 8..60 opsz with font-feature-settings 'tnum','lnum' (titles, settings values, dock numerals at varied weights, dir names) + Source Sans 3 300–700 with font-variant-numeric 'tabular-nums lining-nums' (body, eyebrows, log msgs, all numeric columns).
Rail
five Source Serif 4 numeric tags 1·2·3·4·5 set at varied weights 300/400/500/600/700 — each rail item distinct in stroke weight; a small Source Sans abbreviation (St/Mg/Bp/Db/Lg) sits to the right of each numeral as a faint typographic key, not as a button-secondary-label.
