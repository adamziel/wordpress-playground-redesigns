# Review of #037 — Memphis / 80s

**Direction:** "Memphis / 80s: vivid pinks, teals, squiggle dividers, geometric shapes as decorations."

**Scores:** Feature completeness 9/10 · Direction 10/10 · IA 6/10 · Visual 9/10 · Originality 9/10

## What works
- Memphis-Milano language committed-to without dilution: hot pink, electric teal, mustard, terrazzo confetti backgrounds, chunky 3px black borders, squiggle dividers, Space Grotesk + Space Mono typography. The visual identity is the strongest in the gallery.
- **All 6 §2.2 sources have real inline paste inputs** in the "New" screen:
  - WordPress PR: `<input class="input" placeholder="https://github.com/WordPress/wordpress-develop/pull/…">` (line 1148)
  - Gutenberg PR: `placeholder="github.com/WordPress/gutenberg/pull/…"` (line 1154)
  - From GitHub: `placeholder="owner/repo  ·  branch"` (line 1160)
  - Blueprint URL: `placeholder="https://…blueprint.json"` (line 1166)
  - Import .zip: `<input class="input" type="file">` (line 1172)
  
  This is one of the few designs that completes §3 flows 11–12 with concrete inputs, not just tile faces.
- Apply+Reset destructive copy: "Apply Settings & Reset Playground will **destroy the current site** and rebuild it. Save first if you want to keep your work." (line 1307) with a striped warning band per README.
- Address bar `<input value="/wp-admin/edit.php?post_type=post">` — §2.1 item 2.
- Site Manager renders as a deck of toggleable cards with different geometric motifs — README claims "users orient by texture, not by chrome" which is a clever IA reframe.
- Database tab present; Logs has search input; Files tree has folder rows. Title editor input across multiple states (line 996, 1270, 1317, 1421, 1505) — playground rename consistent.
- Unsaved-Playground notice: "This is an Unsaved Playground. Your changes will be lost on page refresh." (line 1288).
- README is brutally honest: "Memphis maximalism is the opposite of 'calm dev tool' — long sessions will be visually fatiguing... the aesthetic is divisive on purpose; this is the gallery slot for 'playground feels playful,' not the production candidate." Mature awareness of position.
- "Held text on solid blocks to keep WCAG AA" — accessibility considered explicitly.
- Boot state visible at line 1498 with applying-blueprint copy.

## What's missing or weak
- **Apply & Reset has no confirm step.** The striped warning band is great but the button fires. Memphis design vocabulary could turn the confirm into a "are you sure" dialog with geometric shapes blocking the rest of the screen — undelivered.
- **The deck of cards** is described as unifying Settings / Files / Blueprint / DB / Logs into one card-pop pattern — but the artifact also has a "Quick Settings" popover inside the Settings card. Two surfaces for settings remain.
- **Accessibility cost**: README admits "the decorative shapes still buzz" even with text on solid blocks. For users with sensory sensitivities, this design is hard work. The README is honest but doesn't mitigate.
- **No thumbnail preview for blueprints** — README claims "colored swatch because faking screenshots would be dishonest." Fair, but the 43-blueprint discovery cost remains.
- **Chunky 3px black borders eat real estate** — README admits, especially at 390px mobile.
- **Mobile collapses to hamburger + floating Save badge** — described, sensible, but the Memphis decoration likely doesn't translate well to small screens.
- **The "decorative shapes still buzz"** — confetti backgrounds with squiggles + shapes can be busy. WCAG AA is met on text, but visual noise is high.
- **Long sessions will be visually fatiguing** — README admits. A "calm mode" toggle isn't offered.
- **No WP Admin / Homepage explicit chrome buttons** in the toolbar that I can see in the grep — they may be inside the Quick Settings card or as a yellow squiggle overflow.
- **The "yellow squiggle button" for additional actions (Export PR / Download .zip)** is a delightful touch but a first-time user won't parse "squiggle = more actions". Tooltip or label would help.

## How to push harder
- Render the Apply & Reset confirm as a full-screen Memphis-shaped interrupt: a pink geometric overlay with "ARE YOU SURE?" in chunky Space Grotesk, two big buttons (`YES, RESET` mustard / `NO, KEEP IT` teal). Use the metaphor's loudness for the destructive moment.
- Offer a "calm mode" toggle — same layout, neutral palette, no terrazzo. Honor the accessibility cost the README acknowledges without losing the design's identity for those who want it.
