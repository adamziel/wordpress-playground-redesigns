# Review of #074 — Cookbook

**Direction:** "Cookbook: each blueprint is a recipe card with ingredients (plugins, themes), prep time (PHP/WP versions), and a step-by-step 'method'."

**Scores:** Feature completeness 9/10 · Direction 10/10 · IA 9/10 · Visual 10/10 · Originality 9/10

## What works
- Direction faithfully delivered: blueprints rendered as recipe cards with ingredients (plugins, themes, content), prep time (PHP/WP versions, language, network), and a numbered method (blueprint steps). Cookbook spread on home page, "Quick Bites" launcher strip, "Saved Dishes" shelf.
- All 6 §2.2 sources rendered as Quick Bites buttons (lines 868–873): Vanilla WordPress / WordPress PR / Gutenberg PR / From GitHub / Blueprint URL / Import .zip — each with two-line sub-copy ("Boot from a core PR" / "Try a block editor PR" / "Theme · Plugin repo" / "Paste any JSON URL" / "Restore an export").
- **Apply+Reset has best-in-class metaphor framing**: "**Clear the table** Applying these settings will **reset the current playground**. All unsaved data — posts, uploads, configuration — will be wiped, like clearing the bench before the next service." (lines 1183–1184) + Apply & Reset Playground button (line 1186). The flame icon + "clearing the bench" phrasing is uniquely on-metaphor.
- Address bar `<input value="/2025/03/coffee-shop-launch/" aria-label="Inner site path">` (line 735) with aria-label.
- Database (Cellar) carries explicit early-access notice: "The Cellar (Database management) is an **early access feature**. Found a bug? See the GitHub issue tracker." (line 1326). Notably, the metaphor is annotated with the literal label in parentheses — answering the README's claim about "the plain label as a subtitle".
- Logs (Service Log) has search "Search the service log (e.g. 'fatal', 'plugin', '/wp-admin')…" (line 1355). Blueprint has Copy link / Download bundle (lines 1269–1270).
- Unsaved-Playground notice "Unsaved Playground" with "Not on shelf" pill (line 914): "Vanilla WP · scratch session · would be lost on refresh." On-metaphor (saved = on the shelf) + Save site locally mini button (line 1130).
- Playground rename `<input class="titleinput" value="Coffee Shop · Espresso Bar" aria-label="Playground title">` (line 1106).
- The Kitchen tabs renamed (Pantry / Larder / Recipe / Cellar / Service Log) — but README's promise of "plain label as subtitle" is delivered (e.g., "The Cellar (Database management)").
- Additional actions "Plate up & take away" framing (per README) with Download .zip + Export PR.
- README is honest: "A skeuomorphic recipe metaphor risks feeling twee for power users who want speed; I keep the chrome small and the typography legible to compensate, and the toolbar is still one row."

## What's missing or weak
- **Apply & Reset has no confirm step.** The "Clear the table" copy is the strongest metaphor in the gallery, but the button fires. A "type the dish name to send it back" gate would match the metaphor.
- **Settings appear in a popover (line 766) AND the Pantry tab** — two surfaces.
- **The PR / GitHub / URL Quick Bites buttons** have sub-lines but no inline paste inputs — verify whether clicking opens a paste form.
- **The metaphor naming (Larder / Cellar / Service Log)** is mitigated with plain-label subtitles per README — line 1326 confirms "The Cellar (Database management)". But verify all five tabs do this consistently.
- **Long-term learning curve** — README admits "terms like 'Cellar' for the DB might add to the learning curve."
- **The serif display font costs a Google Fonts request** — README admits.
- **Originality overlaps with #073 (Restaurant menu)** — both food/menu metaphors. #074's distinguishing move is the recipe-card structure (ingredients/prep-time/method) versus #073's menu-sections + sommelier. #073 puts the metaphor in the menu copy only; #074 lets the recipe structure shape the blueprint editor itself. Both are excellent.
- **Mobile collapses cookbook spread to single column + bottom sheet for Kitchen** — described.

## How to push harder
- Implement the "Clear the table" as a typed-confirm gate — "Type 'clear the table' to send the dish back. There's no undo, like a real service." The metaphor's existing copy is unmatched; gate it.
- Make sure every Kitchen tab carries the "(plain label)" subtitle the README promises — currently only the Cellar shows this. Larder / Recipe / Pantry / Service Log should each have their literal name visible for new users.
