# Review of v5 #041 — Ventura sidebar × Win11 generous whitespace × fullscreen toggle

**Parent direction:** "Ventura sidebar × Win11 generous whitespace × fullscreen toggle. 5 categories only. Live Preview category has a giant '↗ Open fullscreen' button."
**Affordance count at idle:** 9 honestly (sidebar search + 5 category rows + saved-pill + URL pill compound + Fullscreen CTA) or 8 if you count sidebar search as part of the nav primitive. README rates 4/5 honestly. Over the strict cap but README defends as "the only 'extra' is the very thing the direction demanded."

**Scores (out of 10):**
- Feature completeness: 10 — 9 states (A–I): Live Preview idle, Sites & Blueprints with 6 sources + 4 saved + 20-card gallery + 10 filters, Manage Site Settings + Apply&Reset typed phrase, Manage Site Files + Database, Blueprint with valid JSON, Logs with 14 lines (fatal + stack), Booting (sidebar persists), Fullscreen (1 affordance only), mobile with 5-icon bottom tab bar.
- Restraint: 8 — 9 affordances at idle is over cap, but defended. The Fullscreen CTA is itself the direction's *required* element.
- Iframe primacy: 10 — Live Preview is the default; iframe is the hero; Fullscreen is a 64 px primary button + F. **State H closes the v1 #15 fullscreen gap explicitly** with a glass exit pill + Esc.
- One-door: 10 — Settings popover + SM Settings tab + info-notice Save all collapse into Manage Site. Additional-actions kebab → Manage Site Export/Share. WP-Admin/Homepage live ONLY in Manage Site (typing `/wp-admin` in URL pill still works per brief §2.1). Save + saved-state pill are the same titlebar element.
- One-secondary: 8 — README admits 4/5: "the sub-mode chip strip inside Manage Site is itself a small nested navigation primitive." Same tabs-inside-pane trade as #005/#008/#016.
- Direction adherence: 10 — 5 rows exactly. Ventura tinted glyphs. Win11 Mica + Fluent cards + breadcrumbs + generous whitespace. Giant Fullscreen CTA. **Each parent's signature is present and the *combination* is the design.**
- Visual fidelity: 9 — Mica gradients, frosted rounded cards, oversized whitespace, pill chips, real Ventura-style left sidebar with tinted glyphs, real Win11-style Mica backdrop with radial gradient.
- Synthesis with v1 lesson: 10 — **explicitly combines parents v1 #14 (Ventura sidebar) + v1 #15 (Win11 generous whitespace) + adds the fullscreen toggle to fix the v1 #15 gap.** Cites *seven* prior v5 reviews:

## What works (specific)
- **The "Open Fullscreen" giant CTA is the direction's literal centerpiece**, sized at 64 px (oversized for a button) inside the Live Preview category. The v1 #15 fullscreen gap isn't just *closed* — it's *promoted* to the dominant visual element.
- **Five sidebar categories, exactly** — trimmed from v1 #14's twelve. This is the README's most explicit synthesis claim: "Where v1 #14 sprawled to **twelve** sidebar rows and v1 #15 forgot to give the iframe an explicit fullscreen mode, v5 #041 trims to exactly **five** categories."
- **Seven review citations** with specific implementing elements. Among the highest synthesis density in v5 (matches #018, #025, #026, #029, #031, #035, #036).
- **Sidebar enforces radio-button semantics** (one row active at a time) — review #022's exclusive-open lesson applied structurally, not via policy.
- **The mobile sidebar rotates 90° to a 5-icon bottom tab bar** — same IA, no hamburger collapse. Cleanest mobile-from-sidebar in v5.
- **Typed-phrase confirm card** (review #030 lesson) with Confirm button `disabled` until phrase matches exactly.
- **Save + saved-state pill are the same titlebar element** — review #003 lesson (single embodiment of Save).
- README's honest 4/5 on idle affordance count with the defense ("the only 'extra' is the very thing the direction demanded") rather than fudging the count.

## What's weak or missing (specific)
- **9 affordances at idle is over the strict cap.** README rates 4/5 honestly. The defense is the Fullscreen CTA is required; the sidebar search is "part of the nav primitive." Reasonable but a strict critic dings.
- **The iframe is ~460 px tall** at 1440×900 due to the giant Fullscreen CTA eating vertical pixels. README admits this is the direction's centerpiece *and* the cost. Esc/F gives full edge-to-edge on demand.
- **Manage Site sub-modes are pill chips** (Files / Database / Settings / Export) — tabs-inside-pane trade. README rates one-secondary 4/5 honestly. The defense ("sub-modes share the same scope and never need to be open simultaneously") is the same as #005/#008/#016. Soft seam.
- **Mobile sources grid drops from 3 columns to 2** and "seg filter scrolls horizontally" — README rates 4/5 honestly. Real concessions but small.
- **Body's radial-gradient backdrop is decorative** (README notes -1 on glass critique) — though it's the Win11 Mica signature. Trade between "Mica is the visual idiom" and "no decorative glass."
- The **Fullscreen CTA being a 64 px primary button** is the design's biggest stylistic risk. For a power user who never uses fullscreen, that's a permanent giant button consuming pixels. README defends as direction-required.

## How to push harder (1–2 pointed suggestions)
1. **Make the Fullscreen CTA *also* a saved-state indicator.** Currently it's a single-purpose button. Embedding the saved/unsaved status as a small dot on the CTA (or coloring the CTA's accent) would let it earn double duty — saved-state status + fullscreen action — recovering one affordance slot. Idle count drops to 8.
2. **Promote *one* Manage Site sub-mode to a sixth sidebar row.** Either Files (high-frequency for theme/plugin authors) or Settings (high-frequency for everyone). The remaining sub-modes stay inside Manage Site. This converts the "tabs-inside-pane" trade into "tabs only when the user is in the rarer modes" — a one-secondary-surface improvement with no idle-affordance cost (6 rows + saved + URL = 8).

## Lesson for the next designer
If your direction synthesizes *multiple* parent v1 winners, name them by number in the README's first paragraph and *quantify the trim*: "Where v1 #14 had 12 rows and v1 #15 lacked fullscreen, mine has 5 rows + adds fullscreen as the dominant CTA." The numerical comparison is the synthesis evidence — don't make the critic count.
