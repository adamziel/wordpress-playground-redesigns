Corner FAB pie.

One dot in the corner. That's the chrome. Click it and an eight-slice pie radiates the entire Playground UI — every feature, every tab, exactly one ring outward from the same fingertip. Everything beyond an instant action drops into a single right-side Inspector that swaps content. The iframe is the document; the dot is the menu; there is no third thing.

Idle affordances: 1 (the dot) Pie slices: 8 Secondary surfaces: 1 (the Inspector, 5 swap views) Iframe coverage at idle: 100% Fullscreen toggle: pie slice + F / Esc
1 Idle — desktop 1440×900 · The dot is the only chrome. One affordance.

No top bar, no sidebar, no breadcrumbs. The iframe is the entire viewport. The saved-state lives inside the dot itself — a single colored core (amber = unsaved, green = saved, blue = booting). The chevron hint fades after first interaction.

Pie slice map · 8 → all features

  1. 1 · Refresh — instant. Reloads iframe. (During boot: Cancel.)
  2. 2 · Navigate URL — opens omni-input (paths, pr:#, gpr:#, gh:repo, bp:slug, raw URL).
  3. 3 · Sites — opens Launcher (Start new · Blueprints · Your saved).
  4. 4 · Site Manager — opens Site Manager (Settings · Files · Blueprint · Database · Logs).
  5. 5 · Save — instant. Persists to OPFS. Dot turns green.
  6. 6 · Settings (quick) — opens Site Manager pinned to Settings tab.
  7. 7 · Fullscreen — toggles iframe to true fullscreen. F / Esc shortcut.
  8. 8 · More — Export PR · Download .zip · Share link · wp-cli console · Report bug.

Slice 4 and slice 6 share a destination — same Inspector, different default tab. This is "one door, two doormen," not duplication. The popover, the sidebar tab, and the info-notice (the v1 triple-entry sin) are all the same slice 6.

Self-critique · v5 rubric

[x] Affordance count at idle ≤ 8?           Score: 5/5  (1 affordance: the dot. Saved state is the dot's color.)
[x] iframe primary + fullscreen toggle?     Score: 5/5  (iframe is 100% of viewport at idle; F/Esc + slice 7.)
[x] One door per feature (no duplicates)?   Score: 4/5  (Slices 4 & 6 open the same Inspector, different default tab — same room, two doormen. -1 for that.)
[x] One secondary surface (not many)?       Score: 5/5  (One Inspector. Launcher, Site Manager, URL, More are all overlays from the same FAB.)
[x] Direction expressed strongly?           Score: 5/5  (The entire IA is one dot + one pie. Nothing competes.)
[x] Mobile feels like wider-mobile desktop? Score: 4/5  (Same dot, same pie, same Inspector. Inspector is full-bleed on mobile rather than split. -1 for the split-vs-full divergence.)
[x] Critique pass on glass/ambient use?     Score: 4/5  (Inspector uses glass to let iframe color bleed through; the pie uses near-opaque #0d1220 because hover legibility matters more than transparency. No decorative blur.)
