WordPress Playground · v5 #013 · One Floating Capsule

All persistent chrome lives in a single rounded capsule that floats above the iframe. It carries exactly what is non-negotiable for active use: refresh, URL, save-state name pill, menu. Everything else — Launcher, Site Manager (Settings · Files · Blueprint · Database · Logs), Blueprint gallery, fullscreen, command palette — is summoned from the ⋯ menu and renders into one swappable Stage. Mobile renders the same capsule with the name pill hidden; the Stage becomes a bottom sheet. Press F for fullscreen iframe, Esc to exit.
