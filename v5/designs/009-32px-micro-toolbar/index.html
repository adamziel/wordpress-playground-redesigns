32px Micro-toolbar Only

The entire persistent chrome is a single 32-pixel strip — WP mark, URL, name, save state, sites, menu, fullscreen. Everything else (settings, files, blueprint, database, logs, the launcher) is summoned as a contextual sheet that flies in over the iframe. Press F for fullscreen; press Esc to dismiss any sheet.

L

Affordance ledger.

Counted at idle (Scene 1, no sheets open).
Always-visible affordances at idle — 7 of 8 budget
  1. Refresh — reload iframe (keyboard R)
  2. Address bar — combobox: type a path or pr:, gh:, blueprint: URL schemes
  3. Name pill — shows saved/unsaved state; click to save or rename
  4. Sites chip — opens the Sites sheet (launcher + library)
  5. Site Manager toggle — opens the secondary surface (panel-right icon)
  6. Fullscreen — iframe-only mode (keyboard F, Esc to exit)
  7. Menu kebab — Export to GitHub PR, Download .zip, Cmd+K, Docs, etc.
The WP logo is decorative (links to homepage, not counted as a chrome affordance). Inside any sheet, additional affordances exist — but they're explicitly summoned, not idle.
