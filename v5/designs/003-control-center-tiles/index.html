The grid is the secondary surface One single surface — the top-right quadrant — swaps content. No sidebars. No drawers. Tiles act as both nav and live status (badges, meter fills, toggle states).
Tile language Big tile = primary action. Small tile = panel entry. Toggle tile = boolean. Slider tile = stateful primary verb (Save). The Save tile carries the primary verb across the whole UI.
Long-press A tile briefly press-holds, then animates open into a focused sheet within the same overlay box (see State 3 below). Tapping (short press) jumps to the most-recent state of that surface.
