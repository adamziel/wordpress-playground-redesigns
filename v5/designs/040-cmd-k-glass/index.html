F
Glass tint
Compare state A (Coffee Shop, warm tint — capsule borrows the cream/brown) with K3 (Gaming News, cool tint — capsule borrows the neutral dark). This is "content-aware tint" (v5 §34): three sampled accent colors map to three glass palettes (warm/cool/sage). The blur level (40px) and saturation (170%) stay constant — only the base RGB shifts. Glass earns its keep by signaling which site is beneath, even at idle.

Mobile parity
The desktop palette and the mobile palette are the same React component — only the dock changes from centered card to bottom-attached sheet. Mode chips are the primary nav on mobile (no ⌘ key on phones); slash-typing still works because it's the same input field. Status capsule shortens to "Unsaved" on narrow widths but the LED, separator, and tap-hint stay.
