16 Affordance ledger Hard count of what's on screen at idle, and which feature lives behind which scheme. brief check

Idle affordances · count = 3

  1. State pill — playground name + save status. Clickable: focuses omnibox; if unsaved, also exposes inline Save button.
  2. Omnibox input — the entire IA. Reads URL, schemes, commands.
  3. Fullscreen toggle — single icon button on the right edge. Esc to exit.

Everything else is either inside the omnibox dropdown (one secondary surface), or behind a scheme typed into the same input.

Feature → scheme mapping
