WordPress Playground — Three-layer depth

v5 · Design #035 · Direction: Three-layer depth (refining #38 Glassmorphism). Information hierarchy is rendered as literal z-depth. Three planes: L0 the stage (the iframe, sharp), L1 the capsule (slim persistent chrome, lightly frosted), L2 the inspector (Site Manager, medium-frosted, appears on demand), L3 the library (launcher + 43 blueprints, heavily frosted, recedes everything below). Glass earns its keep because it encodes information — blur strength tells you which layer you're looking at and how to navigate back.

PANEL I · Anatomy

How depth maps to IA

One annotated diagram of the four planes, the blur strengths, and the rules. This is the "you have read the brief" page for a critic.

Three layers + a stage + a confirm shade

L0 · STAGE
The iframe
Always sharp. Always present. Scaled to 0.97 / 0.92 when L2/L3 surface.
blur: 0 · z: 0 · scale: 1 → 0.92
L1 · CAPSULE
Persistent chrome
Refresh · URL · Save · Inspector · Library · Fullscreen · Menu. Always visible at idle.
blur: 8px · z: 20 · 7 affordances
L2 · INSPECTOR
Site Manager
5 tabs: Settings · Files · Blueprint · DB · Logs. One pane, swappable content.
blur: 18px · z: 15 · ⌘I
L3 · LIBRARY
Sites + blueprints
6 sources · 3 yours · 43 BPs. One nav rail, three sections, one search.
blur: 36px · z: 25 · ⌘O
L4 · SHADE
Destructive confirm
Apply+Reset only. Modal sheet with typed-phrase confirm + diff.
blur: 2px · z: 50 · rare
1Depth encodes purpose. Sharper layers = closer to "this is what I'm doing right now." Heavier blur = "this is where I came from / where I can go back to." Blur is not decoration.
2One door per feature. Settings live only in L2 → Settings tab. The L1 capsule has no settings popover. The L3 library has no settings entry. The info-notice button still exists, but only inside the L2 Settings tab.
3One secondary surface. L2 is the only secondary surface — its 5 tabs swap content inside the same pane, the way iPad multitasking does. There are no other side rails, no other drawers, no other docks.
4One primary verb. Save. It lives on the name pill, only when the state is "unsaved." The verb is invisible once saved.
5Depth gauge. The right-edge 4-pip gauge is the design's signature ambient indicator: it shows which layers are currently active. It doesn't add affordances; it labels them.
6Glass earns its keep. Each surface's blur strength is functional, not decorative — it maps to z-order. No frosted decoration appears anywhere else in the design.
7Lessons applied: review #001 (show the morph — panels A→B→C demonstrate it); review #002 (no stacked sub-sections — Library has a real nav rail, not 3 stacked sections); review #006 (didn't smuggle in clever ideas — no Cmd+K, no spatial dock, no traffic-light dots); review #013 (Frequent → Export → Help ordering in the kebab); review #018 (cite reviews by number).
