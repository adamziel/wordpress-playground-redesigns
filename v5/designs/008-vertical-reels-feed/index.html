#008V5 · Vertical Reels Feed

Three pages. One swipe.

WordPress Playground reimagined as a vertical reel. Your live site, a blueprints feed, and your saved playgrounds are stacked — swipe up or down (or scroll-wheel, or arrow keys) to move between them. The iframe is the hero on page 1; every other surface earns its place by being the *content* of a reel, not a competing panel. One secondary sheet for Settings · Files · Blueprint · Database · Logs. One primary verb: Save. One fullscreen toggle. That is the entire chrome.

07State map — how the three reels connect
Three reels (Site / Discover / Your Sites) × three modes (idle / sheet-up / fullscreen) × one boot overlay. Same chrome top-and-bottom regardless of which reel you're on — only the middle changes.
Vertical reels
↑ 1 · Site — iframe is hero. Save/refresh/URL above. Fullscreen toggle right.
swipe ↓ · ⌘↓
↑ 2 · Discover — 43 blueprints as a vertical reel. Search top, up-next preview bottom. Press G for grid.
swipe ↓ · ⌘↓
↑ 3 · Your Sites — saved playgrounds as a feed + dock. "Start new" pulls up the six-source sheet.
One secondary sheet
Press M or drag up "More" → frosted sheet rises over the iframe. Five tabs swap content inside the same sheet:
  • Settings — runtime + capabilities + Apply/Reset (typed-confirm)
  • Files — tree + code editor
  • Blueprint — JSON editor + Run · Copy link · Download
  • Database — SQLite browser, early-access
  • Logs — PHP error log with level filter
Esc closes. Same sheet at 390 px wide: tabs become a horizontal scroller.
Affordance budget · idle
  1. Reel name + saved badge (in top bar)
  2. URL field (in top bar)
  3. Refresh icon (in top bar)
  4. Save pill (in top bar — primary verb)
  5. Fullscreen toggle (top-right)
  6. Reel rail · 3 pips (right edge)
  7. "More" handle (bottom centre)
  8. WP Admin / Home (inside sheet only — not visible at idle, so it's 7 affordances)
7 affordances at idle · within budget

Why the reel?

The TikTok generation already learned this gesture. By giving each of the three landmark surfaces (site / discover / yours) the dignity of a full screen — and stitching them with one swipe — we destroy the triple-entry-point problem. The launcher and library aren't "drawers" anymore; they are pages. The iframe doesn't compete with sidebars; it is page 1. The settings, files, blueprint, db, and logs collapse into a single frosted sheet you raise on demand. The result: one chrome, one verb, one secondary surface, three pages of content. The mobile design isn't a fallback — it's where the metaphor was born.
