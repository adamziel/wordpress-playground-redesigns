044 — Epic launcher × Recede

A cinematic Epic Games-style launcher when there is no live site to interrupt — and a quiet 56 px rail the moment one boots. The launcher is not a panel to open; it is the chrome until a session begins, then it physically recedes to make room for the iframe. One primary verb (LAUNCH) cross-fades into one save-state pill. One secondary surface (the bottom dock) swaps between Settings · Files · Blueprint · Database · Logs. Same primitives at 1440 × 900 and 390 × 844.

Mode A Cinematic launcher

Shown when no playground is running. Full-screen storefront: hero · 6 new-playground sources · 5 featured blueprints · Library of saved sites · 43-blueprint browser. Press LAUNCH or pick a card.

Mode B Receded — running

The launcher collapses to a 56 px left rail. The iframe takes 93 % of the viewport. ≤ 8 always-visible affordances. The bottom dock is hidden at idle. Click rail → launcher rises again.

Scene 1 · Mode A · Cinematic launcher

First-touch, returning user with no active site, or "back to launcher" from a running session. The Epic-storefront IS the chrome. Note: even in Mode A the 56 px rail is faintly visible at the left — it's the bridge between modes (clicking it from inside a running session opens this view; the rail's icons stay constant so the user never loses their bearings).
