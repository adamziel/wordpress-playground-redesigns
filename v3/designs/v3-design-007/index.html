Site Manager actions that do not need a persistent sidebar.

Apply settings and reset Playground?

This will erase the current WordPress database, uploads, plugin edits, theme edits, and unsaved files. The safer default is to save locally first.

Type is not required in this prototype, but the destructive action is deliberately interrupted with a clear cancel path.
