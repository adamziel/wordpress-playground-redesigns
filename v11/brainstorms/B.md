# v11 Brainstorm — Session B

Dark-theme visual directions for the locked v7 #010 interaction model. All comply with: dark BG (#0A–#1A), no purples, no monospace outside code editor, distinctive typefaces, 5 hand-crafted rail marks (Forest Botanical brief), no icon proliferation, single accent, no nested borders, no secondary text under buttons, clarity over decoration.

The rail always has five hand-set marks for **Si**tes, **SM** Site Manager, **Bp** Blueprint, **Db** Database, **Lg** Logs (or analogous glyph per direction). Single quiet identity mark above. No bottom-stacked gear/help/avatar.

---

### 1 — Observatory Readout
- **Visual idiom**: an astronomical control desk at 2 a.m. — flat dark, faint horizon rule, single warm-amber star.
- **Background palette**: rail `#0B0E12` · pane `#10141A` · iframe area `#070A0E`.
- **Text palette**: ink `#E8E2D2`, secondary `#7A8290`.
- **Single accent**: warm amber `#E5A24A` — used only on the active rail dock's 2px left bar and the Saving pill dot.
- **Typefaces**: **Fraunces** (headline, low contrast cut) + **Public Sans** (body). Fraunces gives the antique-instrument feel; Public Sans keeps form labels honest.
- **Rail button treatment**: each dock is a tiny celestial mark in Fraunces small caps + a hand-drawn glyph: Sites = ringed dot, Manager = "SM" italic ligature, Blueprint = a transit-line dash with a tick, Database = three stacked horizon bars, Logs = a Morse triplet. No box around them.
- **Border vocabulary**: 1px `#1B2028` hairlines; pane has one outer border, inner sections separated by a single 1px bottom rule, never a card-in-card.
- **Corner widget shape**: a horizontal "magnitude bar" — site name set in Fraunces italic, tiny amber dot, one stat line.
- **Pane content philosophy**: 5-dock (one pane each). Observatory consoles have dedicated readouts per instrument — no tabs to hunt through.
- **Why it helps focus**: the screen reads like one calm dial array; nothing competes with the iframe.
- **Risk**: amber starts feeling decorative if reused on more than the two permitted spots.

### 2 — Berlin Tape Loop
- **Visual idiom**: a 1980s ECM record sleeve — near-black with a single ivory bar across the top of each pane.
- **Background palette**: rail `#0F0F10` · pane `#141416` · iframe area `#08080A`.
- **Text palette**: ink `#EDE9DD`, secondary `#7C7A74`.
- **Single accent**: cold steel `#9FB2BF` — used only on the active dock indicator and the save-pill text in `Saving`.
- **Typefaces**: **Söhne Breit** alike — fall back to **Mona Sans** (headline) + **Sora** (body). Mona Sans for ECM-cover spacing, Sora for the slight technical edge.
- **Rail button treatment**: each dock is a single set-cap initial pair (Si, SM, Bp, Db, Lg) in Mona Sans medium, tracked at +60, ink secondary. The active one becomes ivory and gets a 1px ivory underline 6px below the cap line. No frames.
- **Border vocabulary**: zero borders. Surfaces separate by background tone shift and a single 1px ivory rule at the head of the pane (the ECM ruled line).
- **Corner widget shape**: an unboxed two-line label, the site name as a typeset header.
- **Pane content philosophy**: 5-dock. Each pane is one page of liner notes — long-form, ruled, scannable.
- **Why it helps focus**: removing every box leaves only language; the iframe wins by being the only thing with edges.
- **Risk**: borderless surfaces can read as a content-management blog if rhythm slips.

### 3 — Tokaido Lacquer
- **Visual idiom**: lacquered black with one rusted vermilion seal — the restraint of a Japanese product manual.
- **Background palette**: rail `#0A0B0C` · pane `#13151A` · iframe area `#06070A`.
- **Text palette**: ink `#E9E4D9`, secondary `#7B7468`.
- **Single accent**: vermilion `#C24A2C` — only on the active dock's 4×4 square seal mark and the saved-state pill.
- **Typefaces**: **Newsreader** italic for the identity & pane titles, **Plus Jakarta Sans** for body. Newsreader's calligraphic tilt is the seal-script echo.
- **Rail button treatment**: a column of small woodblock-style glyphs hand-built in SVG: Sites = a horizon line over water, Manager = a key, Blueprint = a folded paper crease, Database = a stacked-stone cairn, Logs = a single brush stroke. Each is unique, no surrounding shape.
- **Border vocabulary**: none. Sections separated by a 1px `#1F1B17` rule with a 4px vermilion square at the leading end (only on pane headers, used once per pane).
- **Corner widget shape**: a small unframed two-line stamp — site name in Newsreader italic, status word below in Plus Jakarta small caps.
- **Pane content philosophy**: 5-dock. Each tool gets its own quiet page — lacquerware sells discipline.
- **Why it helps focus**: the only red on the screen is wherever your attention should be, and nothing else even has a frame.
- **Risk**: hand-cut SVG marks must be drawn with real care or they read amateurish.

### 4 — Subterranean Telecom
- **Visual idiom**: a Cold-War switching room readout — graphite, phosphor-adjacent green, all light comes from inside the panel.
- **Background palette**: rail `#0C0F0E` · pane `#11161A` · iframe area `#070A0B`.
- **Text palette**: ink `#D8E2DA`, secondary `#6A7872`.
- **Single accent**: signal teal `#3E8A78` (NOT phosphor green — muted) — active dock 2px left bar; nothing else.
- **Typefaces**: **Recursive Sans** (linear, mono-derived but proportional) headline + body. One face does the whole thing — its variable axes let labels feel chiseled.
- **Rail button treatment**: 5 hand-cut "patch labels": each is a Recursive uppercase pair (SI / SM / BP / DB / LG) with a 2-character serial code typeset to the right at half size in a darker tone — looks like a switch panel engraving, not an icon.
- **Border vocabulary**: a single 1px outer pane border `#1A2120`; inside, sections separated by a single bottom rule. Never a nested box.
- **Corner widget shape**: a horizontal engraved bar — "RUNNING · WP 6.7 · 14.2 MB" set with bullets, no box, just a 1px underline.
- **Pane content philosophy**: 5-dock. The metaphor demands instruments be individually labeled.
- **Why it helps focus**: monolithic palette and engraved labeling means there's nothing pretty to look at except the iframe.
- **Risk**: teal could drift toward "tech demo" if it appears anywhere else.

### 5 — Monastery Scriptorium (Nightwatch)
- **Visual idiom**: a hand-illuminated manuscript at candlelight — dark vellum, single gilt initial per item.
- **Background palette**: rail `#0E0C09` · pane `#15120D` · iframe area `#0A0805`.
- **Text palette**: ink `#E6DCC2`, secondary `#7C7058`.
- **Single accent**: gilt `#B3893D` — only on the active dock's hand-drawn initial fill and the save-state pill.
- **Typefaces**: **Cormorant Garamond** (display, only on rail initials & pane titles) + **Atkinson Hyperlegible** (body). Cormorant for the manuscript feel, Atkinson because accessibility is the only excuse for sans in a scriptorium.
- **Rail button treatment**: each dock is a single illuminated capital — **S, M, B, D, L** — set 28px Cormorant with a hand-drawn quill flourish in SVG curling around the active one. No box, no icon, no tooltip-need.
- **Border vocabulary**: a single 1px `#2A2418` ruling line at the head of each pane. No card outlines anywhere.
- **Corner widget shape**: a small inked footer mark — site name italic, dot status.
- **Pane content philosophy**: 5-dock — each tool is its own chapter page, headed by its illuminated initial.
- **Why it helps focus**: the rail looks like a page index, not controls. Reduces cognitive load to "pick a chapter".
- **Risk**: serif display on a dark BG can feel pretentious if leading isn't generous.

### 6 — Mid-Century Playbill
- **Visual idiom**: a Saul-Bass-meets-Push-Pin theater programme — dark stage curtain background, hand-set numerals, single ochre.
- **Background palette**: rail `#101012` · pane `#16161A` · iframe area `#0A0A0D`.
- **Text palette**: ink `#F0E8D8`, secondary `#807A6E`.
- **Single accent**: stage ochre `#D8A23B` — used only on the act number flagging the active dock and the save pill.
- **Typefaces**: **Bricolage Grotesque** (headline) + **Spectral** (body). Bricolage's variable widths give the playbill swagger; Spectral keeps body legible.
- **Rail button treatment**: each dock is numbered like a programme — `I. Si`, `II. SM`, `III. Bp`, `IV. Db`, `V. Lg` — Roman numeral hand-set in Bricolage, label in Spectral italic small. The active item's Roman numeral turns ochre. No icons.
- **Border vocabulary**: a single 1px outer pane border. Internal divisions use a 1px hairline rule only, no nesting.
- **Corner widget shape**: a "Now Playing" cartouche — italic site name, dot.
- **Pane content philosophy**: 5-dock. Five acts, five pages of programme notes. Tabs would break the metaphor.
- **Why it helps focus**: numbering creates a felt-order; users always know where they are.
- **Risk**: italic body type can fatigue on dense forms — keep settings rows generous.

### 7 — North Sea Chart
- **Visual idiom**: a nautical depth chart at night — slate blue-black, single bone-white line, faint isobath grid behind the iframe area only.
- **Background palette**: rail `#0B1116` · pane `#10171E` · iframe area `#070C11`.
- **Text palette**: ink `#E0E4E8`, secondary `#6D7882`.
- **Single accent**: bone `#E6DDC8` — used only on the active dock's leader line and the save pill text (NOT a color — a tonal lift).
- **Typefaces**: **Manrope** (headline, drawn cleanly) + **Lora** (body). Lora for the chart-text feel; Manrope for the geodetic crispness of stat labels.
- **Rail button treatment**: each dock is a tiny **bearing card** — a Manrope two-letter code over a 1px tick mark and a tiny degree number (e.g. `Si · 045°`, `SM · 090°`, `Bp · 135°`, `Db · 180°`, `Lg · 225°`). The degree number is whimsical typography, not data, but it makes every button visually distinct.
- **Border vocabulary**: a single 1px outer border `#1A222A`. Inside, sections separated by faint bottom rule only.
- **Corner widget shape**: a small "fix" — site name with a tiny coordinate-style status line.
- **Pane content philosophy**: 5-dock. A chart has separate panels for separate depths.
- **Why it helps focus**: every label has a unique signature, so muscle-memory builds fast.
- **Risk**: the degree codes are decorative — must stay tiny and one tone darker, or they become noise.

### 8 — Jazz Sleeve (Blue Note Midnight)
- **Visual idiom**: 1960s Reid Miles sleeve — heavy near-black, oversize cropped numerals, one cool foil.
- **Background palette**: rail `#0A0A0B` · pane `#111114` · iframe area `#060607`.
- **Text palette**: ink `#EFEBDD`, secondary `#787268`.
- **Single accent**: cool steel-blue `#7AA0B3` (no purple drift, no cyan glow) — only on the active dock's left-bar and saving pill dot.
- **Typefaces**: **Space Grotesk** (display) + **DM Sans** (body). Space Grotesk gives the sleeve's chunky cropped letterform, DM Sans recedes for body.
- **Rail button treatment**: each dock is a large 22px Space Grotesk numeral 01–05 with a hand-set 8px label beneath it inline (still a single semantic word — `Sites`, etc.). The numeral is what catches the eye; the label is a footnote, not "secondary text" — it's part of the same set-piece.
- **Border vocabulary**: no borders. Pane separates from iframe by background tone only. One single 1px rule under the pane header.
- **Corner widget shape**: a typeset block — site name big, status small below, no box.
- **Pane content philosophy**: 5-dock. Each "track" is its own side of vinyl.
- **Why it helps focus**: oversize numerals act as a visual anchor — eye jumps to dock without searching.
- **Risk**: the small label beneath the numeral could read as banned "secondary text". To stay safe, drop the label and rely on numerals + tooltip only. (Flagging honestly.)

### 9 — Botanical Press (Nocturne)
- **Visual idiom**: a herbarium specimen sheet inverted — black mounting paper, ivory script labels, one pressed-leaf brown.
- **Background palette**: rail `#0D0C0A` · pane `#14130F` · iframe area `#080706`.
- **Text palette**: ink `#E8E1CC`, secondary `#7A7260`.
- **Single accent**: pressed-leaf `#8C6A3E` — only on the saving pill and the active dock's marker line.
- **Typefaces**: **Newsreader** italic (headline / rail / titles) + **DM Sans** (body). Newsreader does the curator's pen.
- **Rail button treatment**: each dock is a single Newsreader italic capital — `S, M, B, D, L` — set 26px with a tiny SVG specimen mark beneath (a leaf, a key, a scroll, a pebble, a feather) hand-drawn at 10px. Each pair is unique; reads as a herbarium tag, not an icon library.
- **Border vocabulary**: pane has a single 1px outer rule. Internal blocks have no borders; they separate by 24px gutter only.
- **Corner widget shape**: an unframed specimen label — site name italic, ink-secondary status.
- **Pane content philosophy**: 5-dock. Each specimen gets its own sheet.
- **Why it helps focus**: the rail looks calligraphic and slow, training the eye to slow down.
- **Risk**: nocturne herbarium can tip toward witchy/precious if any extra ornament creeps in.

### 10 — Brutalist Editorial (Le Monde Diplomatique After Dark)
- **Visual idiom**: a serious newspaper turned night-mode — single ruled column logic, ink secondary headlines.
- **Background palette**: rail `#101112` · pane `#161718` · iframe area `#0B0B0C`.
- **Text palette**: ink `#EDEAE2`, secondary `#7E7B72`.
- **Single accent**: red-ochre `#B14A39` — only on the saving pill text, NOT on the active dock (active dock just goes ink-primary while inactive is ink-secondary).
- **Typefaces**: **Spectral** (headline) + **Public Sans** (body, forms). Spectral is the front-page mast; Public Sans is the agate.
- **Rail button treatment**: rail is a vertical typeset column — each dock is its label set Spectral 12px italic, no icons at all. `Sites / Manager / Blueprint / Database / Logs`. Active item bolds and gains a 2px ink-primary left bar. Inactive items at secondary tone. The "distinct buttons" come from each word being uniquely shaped.
- **Border vocabulary**: single 1px outer pane border. Internal: one rule at the head, no others.
- **Corner widget shape**: a tiny dateline — italic site name, dot.
- **Pane content philosophy**: 5-dock — each tool is its own article.
- **Why it helps focus**: a paper with no graphics teaches the eye to read structure.
- **Risk**: italic-only typeset rail might feel literary rather than navigable. Counter with a clear active-state contrast.

### 11 — Carbon Furnace
- **Visual idiom**: a foundry control terminal — dark slag-black, single molten copper accent that the eye reads as heat.
- **Background palette**: rail `#0F0D0C` · pane `#16130F` · iframe area `#08070A` (very slight warm cast).
- **Text palette**: ink `#E5DED1`, secondary `#7A736A`.
- **Single accent**: molten copper `#C46A2E` — active dock left-bar only + saving pill.
- **Typefaces**: **Sora** (headline, geometric authority) + **Atkinson Hyperlegible** (body). Sora for the headline weight; Atkinson because dark backgrounds wreck legibility unless body type is engineered for it.
- **Rail button treatment**: each dock is a milled "stamp" — Sora 14px uppercase pair (SI / SM / BP / DB / LG) with a single 2px copper notch on the active one's left edge. No outer frame; the negative space around each pair is the button.
- **Border vocabulary**: one 1px outer pane border `#1F1B16`. Internal: a single 1px bottom rule under the pane header only.
- **Corner widget shape**: a horizontal "ingot tag" — site name, status, no frame.
- **Pane content philosophy**: 5-dock — each tool gets its own poured-metal slab.
- **Why it helps focus**: the warm tone is restricted enough that finding it = finding action.
- **Risk**: warm dark BGs can feel uncomfortable for long sessions; tone must be subtle.

### 14 — Cinéma Noir (Title Card)
- **Visual idiom**: a film noir intertitle — full black, italic display, single arc-light bone glow.
- **Background palette**: rail `#08080A` · pane `#101013` · iframe area `#050507`.
- **Text palette**: ink `#EEE8D8`, secondary `#7C7669`.

- **Risk**: very close to #2 Berlin Tape Loop in feeling — risk of duplication. Differs in being warmer and more lyrical.

## Elimination Round

Ranking 1–14 by how well they (a) satisfy every constraint without strain, (b) feel distinct from each other and from v10, (c) help focus rather than decorate:

1. **#13 Atelier Steel** — most disciplined; zero color, one typeface. Cleanest focus.
2. **#2 Berlin Tape Loop** — borderless ECM elegance; the iframe wins by being the only thing with edges.
3. **#1 Observatory Readout** — Fraunces + Public Sans is a memorable pair; instrument-dial metaphor flatters the dock.
4. **#4 Subterranean Telecom** — Recursive Sans is unusual and earns its place; engraved label rail is genuinely fresh.
5. **#12 Inked Schema** — Fig.-N rail is a unique idea, restrained accent, journal feel matches "tool".
6. **#10 Brutalist Editorial** — typeset-word rail is the most literal "no icon" answer; clean.
7. **#3 Tokaido Lacquer** — beautiful if SVG marks are crafted; biggest payoff if executed well.
8. **#5 Monastery Scriptorium (Nightwatch)** — illuminated capitals are striking; Atkinson body keeps it usable.
9. **#6 Mid-Century Playbill** — numbered acts is a nice anchor, but ochre + italic body is two flavors at once.
10. **#7 North Sea Chart** — clever bearing codes but the degree numerals risk reading as noise.
11. **#11 Carbon Furnace** — copper is risky; warm dark BG fatigues.
12. **#14 Cinéma Noir** — close cousin of #2, drop unless we need a lyrical option.
13. **#9 Botanical Press (Nocturne)** — risks the v10 #012 echo; nocturne version of an idea already shipped.
14. **#8 Jazz Sleeve** — only one with a label beneath the numeral; risks the "secondary text" rule even if framed differently.

**If we had 10 slots, ship: 1, 2, 3, 4, 5, 6, 7, 8, 10, 12.**

If we had only 8: drop #6 Mid-Century Playbill (italic body fatigue) and #7 North Sea Chart (decorative degree numerals).

If we had only 6: drop also #3 Tokaido Lacquer (execution-dependent) and #5 Scriptorium (serif-display risk), leaving the 6 most-disciplined: **#13, #2, #1, #4, #12, #10** — a coherent set that all keep the iframe loudest and the chrome literary.
