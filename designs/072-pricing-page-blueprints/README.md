# 072 — Pricing-page-style for blueprints

## Thesis
The 43-blueprint gallery is reframed as a SaaS pricing page: **"Choose your starter."** Each blueprint is a tier-style column with a headline price (PHP/WP combo), a one-line value prop, a vertical checklist of included "features" (plugins, themes, content, demo data), a recommended badge, and a single big primary CTA. A "Compare all plans" table sits below for power users who want to scan the matrix side-by-side. The familiar grammar of Stripe / Linear / Vercel pricing pages lets first-time visitors triage 43 options in seconds — the visual hierarchy does the curation work that a flat 43-card grid cannot.

## What I remixed / merged / removed
**Remixed:** the launcher's three sections collapse into a single scrollable pricing landing — "Start fresh" sources (Vanilla, WP PR, Gutenberg PR, From GitHub, Blueprint URL, Import .zip) become the **Free tier strip** at the top (six lightweight cards before the paid-style tiers, signalling "no recipe, just configuration"). "Your Playgrounds" becomes the **"Resume your subscription"** row — accounts you already "own." The full blueprint gallery + featured 5 merge into one continuous scroll: featured tiles render as oversized highlighted tier columns ("Most Popular", "For Developers", "For Creators"); the remaining 38 fold into the compare-table. **Merged:** the three settings entry points become one Settings popover anchored to the toolbar gear; the Site Manager keeps its 5 tabs but its header is restyled as an account-management sidebar ("Manage subscription") so the metaphor is consistent. **Removed:** the right-side drawer and category filter chips — categories become column groupings in the compare table ("Filter columns") and a left-rail of pricing pivots ("Show: Themes only / WooCommerce / Gutenberg / Experiments"). Logs and DB browser are presented as "Advanced add-ons" tabs inside Site Manager.

## Trade-offs
Pricing tier columns naturally suggest hierarchy and choice anxiety — useful for newcomers, condescending for repeat developers. Mitigated with the Compare table and a keyboard-accessible search overlay. The metaphor breaks if pushed too literally: there is no money, no checkout, no plans-that-lock-features, so the word "Free" appears on every tier (which is honest but redundant). The big tier columns are real-estate hungry on mobile — collapsed to a swipeable carousel with sticky CTA. Last: the pricing metaphor risks making this feel commercial; we lean into Automattic-y warmth (rounded corners, soft shadows, plenty of whitespace) instead of cold enterprise SaaS.
