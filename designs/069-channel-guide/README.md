# #069 — Channel Guide / TV Grid

## Thesis
Playground reimagined as a cable TV box. The iframe is a literal CRT-style "screen" tuned to a channel; every blueprint is a numbered channel and the categories are network groupings. The Electronic Program Guide (EPG) is a 2D grid — rows are categories (Featured, Website, Personal, Content, Themes, Gutenberg, Experiments, WooCommerce, News, Personal Recordings) and columns are individual blueprints scrolling horizontally. You "tune in" by selecting a cell. Channel-up/channel-down via arrow keys. The whole IA collapses around a remote-control mental model: small chrome around the screen, big GUIDE button summons the grid, dedicated buttons for DVR (Saved Playgrounds), SETTINGS, INPUT (Address bar / GitHub / Blueprint URL), and REC (Save).

## What I remixed / merged / removed
The launcher panel and blueprint gallery merged into one EPG overlay — no more separate "Start a new Playground" + "Start from Blueprint" + "View all 43" screens. New-Playground sources (Vanilla, WordPress PR, Gutenberg PR, From GitHub, Blueprint URL, Import .zip) became a top "Input Sources" channel-strip row on the guide so they read as broadcast networks alongside curated blueprints. "Your Playgrounds" became the **DVR Recordings** row at the top of the guide. The Site Manager kept its five tabs but reskinned as the TV's "Settings menu" sliding from the right; the Settings popover collapsed into the same menu so there is one settings surface. Toolbar buttons read as remote-control keys. The "Apply settings & Reset" destructive action gets a CRT-style confirmation dialog.

## Trade-offs
The metaphor is loud — it commits to a phosphor-green-and-amber CRT aesthetic that will polarize. Power-users will love the keyboard-channel-surfing; first-time designers may find the channel-number indirection cute but slow versus a plain card grid. Horizontal scrolling per category row hides the "43 total" count behind motion, mitigated by an "All Channels" virtual row. Mobile loses the 2D grid (it stacks rows vertically and turns the screen into a fullscreen mode with the GUIDE as a bottom-sheet) — that's a deliberate downgrade because EPGs don't make sense on a phone. The Database/Logs/Files tabs reskinned as "engineering channels" lean into the bit, but real PHP work happens in the same code editor — no functionality lost.
