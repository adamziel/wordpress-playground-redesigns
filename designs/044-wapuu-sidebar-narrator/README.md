# Design #044 — Wapuu-centric mascot UI

## Thesis
Wapuu, the WordPress community mascot, gets a permanent home: a slim left-side **Companion Rail**. He doesn't decorate; he *narrates*. As you boot a playground, change PHP versions, save a site, or open the file browser, Wapuu posts a short, calm message into a running chat-style timeline ("Booted on PHP 8.3 — nice and fresh.", "I spotted you editing wp-config — careful."). He also recommends two relevant blueprints on the home state, surfaces a single contextual hint when something is destructive, and otherwise gets out of the way. The rest of the chrome is a clean, professional WordPress-blue UI — Wapuu carries the personality so the working surface doesn't have to.

## What was remixed / merged / removed
The five Site Manager tabs (Settings, Files, Blueprint, Database, Logs) are kept but reorganized into a **right-side workbench** that slides in over the iframe instead of pushing it, freeing horizontal space. The blueprint launcher, Your Playgrounds, and "Start a new" rows merge into one **Launcher overlay** with three columns (Start fresh / Pick a blueprint / Resume), reducing the three current launcher sections to one screen. Settings exist in two places (the popover and the Site Manager tab) — both kept, but unified visually so they read as the same form. WordPress PR, Gutenberg PR, From GitHub, Blueprint URL, and Import .zip all share one "Advanced start" row. Wapuu's narration **replaces the toast region** for low-severity messages but toasts still fire for errors. The "Apply Settings & Reset" destructive button gets a Wapuu warning bubble before firing — a single, dismissable, non-blocking nudge.

## Trade-offs
A mascot-driven UI risks becoming Clippy. The defense: Wapuu's bubbles are short, color-muted (one accent line, no animations on idle), max one bubble visible at a time with a "View timeline" affordance for history, and a one-click **Mute Wapuu** in the rail's footer that collapses him into a static icon. The Companion Rail costs 280px on desktop — significant but recoverable via a collapse toggle (rail becomes a 56px icon strip). On mobile the rail becomes a bottom sheet that pulls up; Wapuu's avatar lives in the bottom-right as a FAB. The biggest risk is that power users will mute him on day one and never see his recommendations — accepted, because the UI still works fully without him; he is a *layer*, not a load-bearing element.
