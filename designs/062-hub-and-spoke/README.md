## Thesis
Playground becomes a mission-control hub. The running WordPress site is literally at the center of a circular console; every major action — Save, Settings, Site Manager, Blueprints, Your Playgrounds, New Session, Files, Logs, Database — is a labeled spoke radiating from that center. Choosing a spoke "docks" it: the hub shrinks into the bottom-right corner and the chosen surface flips open as a full sub-screen. There is one home you always return to, one orbit to scan, and exactly one focused workspace at a time.

## What I remixed / merged / removed
The top toolbar collapsed into the hub's inner ring (Refresh, address bar, name + Save, version chip). The Saved Playgrounds launcher and the Site Manager — historically two separate drawers — are now two sibling spokes on equal footing, each opening into a dedicated sub-screen with the same chrome. The three duplicate settings entry points (popover, sidebar tab, info notice) merged into a single Settings spoke; the inline "Save site locally" notice is rendered as a pulse on the Save inner button. The blueprint browser, File browser, Blueprint editor, Database and Logs each became their own spoke (a five-tab Site Manager would have hidden them). Additional actions (Export GitHub PR, Download .zip), Import .zip, WordPress PR, Gutenberg PR, From GitHub and Blueprint URL are surfaced as launch-pad cards inside the "New" spoke instead of being buried in a dropdown.

## Trade-offs
Radial layouts cost desktop screen area: the hub on the home view dedicates the whole viewport to navigation, so the iframe shrinks to a circular porthole until you click "Enter site". Power users who live in the file tree or logs have one extra click to leave a sub-screen and pick another spoke, versus a persistent sidebar. The orbital metaphor is delightful at 1440 but doesn't survive a 390px viewport, so mobile degrades to a vertical action list with the same labels — same destinations, different shape. Discoverability is the win: every feature has a fixed angular address that never moves.
