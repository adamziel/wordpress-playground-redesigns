Home › Welcome

Hello from WordPress Playground!

You're running a real WordPress, in your browser, with no server. PHP runs through WebAssembly and MySQL is emulated via SQLite. Try refreshing — the site persists for this session.

This iframe is the canvas. Everything around it is the chrome — and in this redesign the chrome looks like Discord. The active server on the far left is this playground; the channel list shows the panels you can inspect; the members panel on the right is your blueprint browser.

Switch to #files to inspect wp-content/themes. Switch to #logs to read PHP error output as a chat stream. Edit the URL above to navigate.
