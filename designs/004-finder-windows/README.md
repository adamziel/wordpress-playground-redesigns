# Design #004 — Mac Finder windows

## Thesis
A WordPress Playground is, conceptually, a disposable computer-in-a-tab. This design leans all the way into that mental model: the entire surface is rendered as a faux macOS desktop. Each saved or running playground is a real *window* with traffic-light buttons, a drag handle, a title, and its own toolbar. Multiple windows can coexist on the desktop (one running site, one Finder window browsing blueprints, one inspector showing files / logs / DB). The familiar Finder vocabulary — sidebar of *Locations*, column view, Get Info pane — does the work that "Site Manager", "Saved Playgrounds drawer", and the settings popover used to do separately.

## What I remixed / merged / removed
The three settings entry points collapse into a single "Get Info" inspector that slides out of the active window. The "Saved Playgrounds" drawer becomes a Finder window with sidebar Locations: **Playgrounds**, **Blueprints**, **New From…** (PR, GitHub, URL, .zip), **Recents**, and a **Trash** for reset. The blueprint gallery is column-view inside that Finder, with categories as folders. Site-Manager tabs (Settings/Files/Blueprint/DB/Logs) become *toolbar segments* on the running-site window, since they're all "inspect this running site" anyway. The macOS menu bar at the very top exposes every feature as a real menu (File ▸ New Playground, View ▸ Show Logs, Window ▸ list of open windows) — this gives the design a complete keyboard-discoverable command surface without inventing a new "command palette" feature. The "Apply Settings & Reset" destructive action is reframed as a system dialog with explicit copy.

## Trade-offs
The Finder metaphor is heavy: users on touch / Linux / mobile may not read the chrome the same way, so on mobile I collapse the desktop into a single full-screen window with a bottom dock (no overlapping windows). Multi-window also implies a layering / window-manager state model that doesn't exist today — I render the overlap statically. Skeuomorphism risks looking dated, so the palette deliberately uses modern Sonoma-era translucency, SF-style typography (Inter as a stand-in), and restrained traffic lights rather than 2008-era pinstripes.
