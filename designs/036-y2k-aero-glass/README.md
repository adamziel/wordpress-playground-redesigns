# Design #036 — Y2K / Aero Glass

## Thesis
This redesign treats WordPress Playground like a desktop OS from the cusp of 2003-2007 — a love letter to Windows Vista Aero, OS X Aqua, the original iPod nano, and Y2K product design (think *iMac G4*, Encarta, MSN Messenger). The browser viewport becomes a glossy desktop; every Playground surface is a draggable glass window with beveled chrome, a chromatic-aberration title bar, and jellybean buttons that look wet. The iframe sits in the largest pane, lit like a screen behind frosted glass.

## What I remixed / merged / removed
The seven toolbar items, the Saved Playgrounds drawer, and the Site Manager all live as **floating Aero windows on a single glass desktop**. Top-of-screen is a unified "Aero glass strip" replacing the toolbar — refresh, URL bar (called "Address Bar" exactly like an XP browser), Playground label, Save, Launcher, Site Manager toggle and Settings popover. The Site Manager's 5 tabs (Settings, Files, Blueprint, Database, Logs) become a single tabbed window pane with chromatic chrome. The Saved Playgrounds launcher merges all six "Start a new Playground" entries (Vanilla, WordPress PR, Gutenberg PR, From GitHub, Blueprint URL, Import .zip) into one row of iridescent jellybean buttons, sitting above featured blueprints and the full 43-card gallery (with category filters + search). "Additional actions" (Export to GitHub PR, Download .zip) live in the Site Manager titlebar gear menu. The settings popover is preserved as a small dock-attached panel — three settings entry points (popover, Site Manager tab, info-notice button) are intentionally kept to honor the brief's "no feature drops" rule, but unified visually.

## Trade-offs
Aero is beautiful but heavy: gradients, bevels, blurs and chromatic accents eat readability and accessibility. To compensate, body type is dark on translucent surfaces and minimum sizes are bumped. The aesthetic also fights modern users' "flat" muscle memory — affordances are slightly louder than they need to be. Performance-wise, real CSS `backdrop-filter: blur()` chains and many radial gradients are taxing; in production I'd composite to a few raster textures. Finally the chromatic aberration accents (red/blue split borders) are decorative rather than functional, but that's exactly the Y2K thesis: surfaces that *feel* like glass and chrome, not minimalist rectangles.
