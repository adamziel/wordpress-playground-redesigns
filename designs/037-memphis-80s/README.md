# Design #037 — Memphis / 80s

## Thesis
Playground is a developer tool that has always tried to feel sober and "professional." This direction throws that out: it dresses every panel of the existing app in the visual language of the Memphis-Milano group and 80s post-modern design — hot pink, electric teal, mustard, squiggles, terrazzo confetti, chunky black borders, no rounded corners except for big circles, and Space Grotesk + Space Mono typography. The aesthetic loudly signals "this is a *playground*, not production." The interface becomes a poster you can click.

## What I remixed / merged / removed
The five Site-Manager tabs (Settings, File browser, Blueprint, Database, Logs) and the Saved-Playgrounds drawer are unified into one **deck of toggleable "cards"** that pop in over the iframe — every card has the same Memphis frame but a different geometric motif, so users orient by texture, not by chrome. The three settings entry points (popover, sidebar tab, info-notice button) are merged into a single **"Quick Settings"** popover that lives inside the Settings card; the destructive Apply+Reset action gets a striped warning band so it can't be missed. The launcher's six "Start a new Playground" choices (Vanilla / WP PR / Gutenberg PR / From GitHub / Blueprint URL / Import .zip) become six oversized chunky tiles on the New screen. The Blueprint gallery's category filter is a row of pill chips; cards carry a colored "swatch" instead of a thumbnail because we can't ship real screenshots. The Additional-actions menu (Export to GitHub PR, Download .zip) lives as a yellow squiggle button in the toolbar.

## Trade-offs
Memphis maximalism is the opposite of "calm dev tool" — long sessions will be visually fatiguing, and accessibility around contrast on confetti backgrounds is a real concern (I held text on solid blocks to keep WCAG AA, but the decorative shapes still buzz). The chunky 3px black borders eat screen real-estate, especially on the 390px mobile breakpoint where the toolbar collapses to a hamburger + the floating Save badge. There is also no thumbnail preview for blueprints — the design intentionally substitutes a colored "swatch" because faking screenshots would be dishonest. The aesthetic is divisive on purpose; this is the gallery slot for "playground feels playful," not the production candidate.
