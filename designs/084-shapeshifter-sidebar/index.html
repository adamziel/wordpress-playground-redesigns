Interaction map
One sidebar, six modes.
Activity bar (44px) picks a mode → entire sidebar transforms. No tab juggling inside.
Modes carry context.
Logs mode has search + level filters built-in. DB mode has table list + grid. Each pane is a real app, not a tab strip.
Launcher is also "a mode".
Same activity icon (bottom of column) — but it overlays the iframe in a sheet, since the user is starting fresh.
Mobile: column rotates 90°.
The activity bar becomes the bottom tab bar. Sidebar becomes a bottom sheet. Iframe is the canvas.
Use the buttons above to switch state and mode. Site · Files · Blueprint · DB · Logs · Settings · Launcher.
