Gutenberg-inside-Gutenberg

The Playground chrome IS the block editor. The running WordPress site is a single selected "Site Block": floating block toolbar above, slide-out inserter on the left (sources, blueprints, saved), tabbed inspector on the right (Settings, Files, Blueprint, Database, Logs). Every Playground feature maps onto a block-editor pattern an existing Gutenberg user already knows.

Mobile — bottom sheet inspector

At 390 px the inserter and inspector collapse behind the top hamburger and the bottom navigation. The block toolbar still sits above the site iframe and stays the primary surface — but the address bar now stretches edge-to-edge and the more-actions menu houses Export PR / Download zip / Copy link.

  • Insert opens the inserter as a full-screen sheet (Blueprints / Sources / Saved tabs).
  • Blueprint opens the JSON editor as a sheet with Run / Copy / Bundle.
  • Inspect opens the Settings + Files + DB inspector as a sheet.
  • Logs is a dedicated tab because debugging on mobile usually means logs.
  • Save status badge stays in the top bar; tapping it persists to OPFS.
